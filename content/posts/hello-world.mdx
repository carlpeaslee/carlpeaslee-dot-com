The first job for a site like this is to stay simple without being limiting.

This version is built around a static-first React Router app that prerenders the public routes at build time, then hydrates client code where it actually adds value.

That means the landing page and blog posts ship as HTML, but the writing format can still host stateful React components when a post needs something more interactive than screenshots.

## Why this stack

- Vite keeps local development fast.
- React Router handles SSR and route-level rendering cleanly.
- Cloudflare is a good fit for globally cached pages and simple deploys.
- Tailwind and shadcn keep the styling system lightweight and composable.
